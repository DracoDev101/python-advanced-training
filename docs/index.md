# Python / Django 生产级后端系统训练营

这是一套面向生产后端工程的 Python / Django 进阶训练资料。

核心覆盖：

- Python runtime、GIL、内存、typing、import 与工程边界。
- Django 项目结构、请求链路、ORM、transaction、migration、DRF/API、缓存、测试。
- Celery broker/backend、worker、幂等、重试、超时、路由、监控与队列排障。
- WSGI/ASGI、Gunicorn、Daphne、Channels、Uvicorn worker。
- greenlet/gevent/eventlet、monkey patch 的收益、污染面与调试方法。
- profiling、observability、部署、runbook 与综合项目。

课程目标不是列 API，而是建立：机制理解 → 代码实验 → 生产取舍 → 故障证据 → 可验证修复 的闭环。
